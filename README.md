# Flutter Timer BLoC

A simple Flutter timer to learn about BLoC pattern following [this tutorial](https://medium.com/flutter-community/flutter-timer-with-flutter-bloc-a464e8332ceb)

## Getting started
- Install dependencies: `flutter pub get`
- Run app `flutter run`
